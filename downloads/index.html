安装指南（当前版本： v0.4.5）

Julia提供以下几种运行Julia的方式：
  • 命令行模式
  • Juno IDE
  • 在浏览器上直接通过IJulia notebook使用的JuliaBox（不需要安装，在浏览器中打开这个地址，并注册即可）

    • 安装包下载：

    Windows用户

    请下载： 32位安装包/64位安装包

    Ubuntu/Debian/Fedora/CentOS/SL 包（rpm）用户

    请将ppa源添加至系统中

    sudo add-apt-repository ppa:staticfloat/juliareleases
    sudo add-apt-repository ppa:staticfloat/julia-deps
    sudo apt-get update
    sudo apt-get install julia

    Mac用户

    请下载 Mac OS X安装包 64位安装包（10.7+）
